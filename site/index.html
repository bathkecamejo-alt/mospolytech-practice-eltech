Главная

Кроссплатформенное приложение Московского Политеха

Проект «Eltech» — это современное кроссплатформенное мобильное приложение, созданное для улучшения цифровой среды университета.

Основная ценность проекта заключается в использовании передовых технологий: Kotlin Multiplatform (KMP) для переиспользования бизнес-логики через единый SDK и Jetpack Compose для создания современного пользовательского интерфейса, соответствующего дизайн-коду Московского Политеха.

О проекте

Обоснование и актуальность

В настоящее время существует проблема отсутствия полноценного мобильного приложения для Московского Политеха, что создает неудобства при использовании Личного Кабинета (ЛК) со смартфонов. Разработка проекта актуальна, так как она значительно повысит пользовательский опыт студентов и преподавателей от взаимодействия с цифровой средой университета.

Цели, задачи и результаты

Цель: Создать полноценное кроссплатформенное мобильное приложение для удобного доступа к информации Московского Политеха со смартфонов.

Задачи:

  • Перенос основного функционала веб-версии ЛК Московского Политеха в формат мобильного приложения.
  • Перенос текущего функционала на архитектуру Kotlin Multiplatform (KMP) для создания общей библиотеки и SDK.
  • Разработка пользовательского интерфейса с дизайном, полностью соответствующим визуальному стилю Московского Политеха.

Ожидаемые результаты: Дальнейший запуск единого приложения для платформ iOS и Android, а также получение приложением статуса официального клиента университета.

Описание и характеристики продукта

  • Кроссплатформенное мобильное приложение «Eltech».
  • В основе архитектуры лежит технология Kotlin Multiplatform (KMP), позволяющая переиспользовать бизнес-логику через единый SDK.
  • Визуальная часть приложения строго опирается на дизайн-код Московского Политеха, чтобы приложение воспринималось как органичная часть цифровой среды вуза.
  • Проект имеет открытый исходный код и представлен на GitHub.

Взаимодействие с организацией-партнёром и Карьерный марафон

В рамках дистанционного участия в карьерных мероприятиях был подробно изучен доклад руководителя службы разработки Яндекс Доставки Александра Голубева «Когда действительно нужны алгоритмы: опыт оптимизации KD-Tree» на технической конференции «Истовый Инженер».

В ходе вебинара были освоены следующие ключевые аспекты, применимые к разработке нашего командного SDK для проекта Eltech:

  1. Теоретическая база пространственного ветвления (KD-Tree) на примере двухмерного пространства (широта и долгота) для поиска ближайших объектов.

  2. Программная реализация алгоритмов обхода с использованием структуры данных std::stack для итеративного обхода узлов и отсечки ложных веток пространства.

  3. Бенчмаркинг и нагрузочное тестирование на специализированном стенде. Анализ перцентилей (50%, 95%, 98%) показал кратное снижение задержек (timings в ms).
