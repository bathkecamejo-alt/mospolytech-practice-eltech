# Участники проекта Eltech и личный вклад
Мобильное приложение «Eltech» является крупным open-source проектом, над развитием которого трудится команда разработчиков Центр компетенций «ИТ-кластер» Московского Политеха. В рамках текущей учебной практики состав рабочей группы выглядит следующим образом:
* Клепиков Станислав (Руководитель проекта) — общая координация, принятие архитектурных решений, управление исходным кодом на GitHub и публикация релизов. Вклад: 50 часов.
* Воронин Даниил (Ведущий разработчик) — проектирование ядра и архитектуры основной библиотеки SDK_Eltech для взаимодействия с внутренними API университета, подготовка базовой технической документации. Вклад: 45 часов.
* Лёх (Инженер по информационной безопасности) — обеспечение безопасности SDK, проектирование механизмов безопасной аутентификации запросов, внедрение алгоритмов хеширования и соли (hash_salt.txt) для защиты персональных данных в базах данных и students.json. Вклад: 30 часов.
