# Журнал прогресса проекта Eltech
## 01.03.2026
* SDK_Eltech: Начата разработка основной библиотеки (SDK) для взаимодействия с внутренними API университета.
* Документация: Подготовлено техническое описание базовых методов SDK, описаны структуры запросов и ответов для обеспечения согласованной работы команды.
## 15.03.2026
* Экран физической культуры: Реализован специализированный экран для отслеживания прогресса по физкультуре. Внедрена логика отображения накопленных баллов и посещений занятий на основе данных из личного кабинета.
## 26.03.2026
* Интеграция с LMS: Реализован базовый функционал взаимодействия с системой управления обучением (LMS/Moodle). Настроены механизмы парсинга учебных материалов и интеграция сессий для бесшовного перехода из приложения к заданиям.
## 15.04.2026
* Доменный слой: Завершена работа над архитектурой Domain Layer. Описаны бизнес-сущности (Entity) и сценарии использования (Use Cases), что позволило отделить логику приложения от деталей реализации инфраструктуры.
## 22.04.2026
* Экран расписания: Реализована базовая верстка основного экрана приложения. Созданы адаптивные UI-компоненты для карточек занятий, реализована логика отображения текущего дня и переключения между датами.
